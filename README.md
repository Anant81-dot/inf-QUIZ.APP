# 🧠 inf-QUIZ.APP

A fun and interactive quiz web app with five different questions — built with pure HTML, CSS, and JavaScript. No frameworks, no libraries — just clean vanilla code.

## ✨ Features

- 5 interactive quiz questions
- Instant right/wrong feedback on each answer
- Score tracking and final result display
- Clean and responsive UI — works on mobile & desktop
- Built with zero dependencies — pure HTML, CSS & JS

---


## 🚀 Getting Started

### Run locally

```bash
# 1. Clone the repository
git clone https://github.com/Anant81-dot/inf-QUIZ.APP.git

# 2. Open the project folder
cd inf-QUIZ.APP

# 3. Open index.html in your browser
# Just double-click index.html — no setup needed!
```
